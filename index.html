Comfy Corp Web App

Onze opdracht wordt het maken van een webinterface voor de SIR 120 PRO radio. Dit interface, of dashboard, kan gebruikt worden voor het instellen van de radio, het opzetten van een alarm en het starten van een stream.
Tijdens deze opdracht zullen we gebruik maken van verschillende technieken besproken onder het kopje technieken.

Functionele eisen

Eis Beschrijving
Kunnen verwisselen van stream Het realtime kunnen veranderen van de stream op de radio.
Het ophalen van een status van de radio Bekijken wat de status is van de radio (gestopt, gestart en welke stream)
starten van een stream het kunnen starten van een stream op de radio
Stoppen van een stream het kunnen stoppen van een stream op de radio
Instellen van een alarm Het instellen van een alarm op de radio
verwijderen van een alarm Het verwijderen van een alarm op de radio
Instellen van een alarm Het instellen van een alarm op de radio
Instellen van een volume Het instellen van een volume op de radio. Deze te kunnen veranderen tijdens het spelen
Een login methode Een manier om radios te registreren en in te loggen met deze gegevens.
Tonen huidig nummer Laten zien welk nummer er wordt afgespeelt op de radio. (Met coverart)
Notification system Tonen in notificatie vorm dat er een nieuwe stream of nummer begint.

Technieken Toepassen.

De volgende technieken zullen worden toegepast

  • HTML5: Basis layout voor de website.
  • CSS3: CSS3 wordt gebruikt voor het stylen van de website.
  • Javascript: Javascript wordt gebruikt voor de interactie van de gebruiker met de interface.
  • jQuery: JQuery wordt gebruikt om data te processen die als Json binnenkomt van de radio.
  • JSON: Gebruikt voor de data van radio te ontvangen in de webinterface.
  • PHP: Er wordt gebruik gemaakt van php aan de kant van de server om te interacten met de radio.
